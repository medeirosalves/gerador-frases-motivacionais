# ✨ Gerador de Frases Motivacionais

Um projeto simples desenvolvido em **Java** que gera frases motivacionais aleatórias combinando diferentes palavras e expressões armazenadas em arrays.

---

## 📖 Sobre o Projeto

O programa seleciona aleatoriamente:

- Uma virtude
- Uma ação
- Um objetivo

E combina esses elementos para criar uma frase motivacional única a cada execução.

### Exemplo

```text
Frase sorteada: A persistência é o primeiro passo para um futuro brilhante.
```
